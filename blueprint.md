
# Service Design Exhibition Website

## Overview

This project is a web-based exhibition showcasing the results of a Service Design class. It will feature a main page with a title, a brief introduction, and a gallery of student projects. Each project will have its own dedicated section with more detailed information.

## Style and Design

*   **Theme:** Modern, clean, and professional.
*   **Color Palette:** A combination of neutral colors (white, light gray) with a single accent color for calls to action and highlights.
*   **Typography:** A sans-serif font for readability, with a more expressive font for headings.
*   **Layout:** A responsive grid-based layout that adapts to different screen sizes.

## Implemented Features

*   **Project Gallery:** A grid of project cards, each with a thumbnail image and a title.
*   **Project Modal:** A modal window that displays project details (title, image, and description) when a project card is clicked.

## Current Plan

The initial version of the website is complete. The project is ready for further enhancements or content updates.
